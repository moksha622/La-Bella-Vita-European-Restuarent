# 🌍 La Bella Vita – European Restaurant Website

A modern, responsive **European Restaurant Website** designed to showcase authentic cuisine, enable table reservations, and support online food ordering with a smooth user experience.

## 📌 Project Overview

**La Bella Vita** is a fully responsive multi-page restaurant website featuring:

- 🍝 European food menu (Italian, French, Spanish, German)
- 📅 Online reservation system
- 🛒 Food ordering & payment flow
- 🔐 User authentication (Login / Signup)
- 🗺️ Google Maps integration for delivery location

This project demonstrates strong **frontend development**, **UI/UX design**, and **real-world web application flow**.
